# 🏠 Sistema Espaço e Arte

Site institucional e sistema administrativo desenvolvido para a **Espaço e Arte – Marcenaria Moderna**, empresa com 15 anos de atuação em Palmas – TO, especializada em móveis planejados, projetos de interiores e assessoria em obras. O projeto contempla um portal público para captação de clientes e um painel administrativo interno para gestão de clientes, serviços e compromissos.
O projeto utiliza o ecossistema Spring, aproveitando a convenção sobre configuração para acelerar o desenvolvimento. A arquitetura segue o padrão MVC (Model-View-Controller) para manter uma separação clara de responsabilidades entre as chamadas HTTP, lógica de negócio e persistência.

    Java 17/21: Linguagem base para a implementação da lógica.
    Spring Boot 3: Framework para criação de aplicações independentes e prontas para produção.
    Spring Data JPA: Abstração para manipulação de dados de forma elegante.
    PostgreSQL: Banco de dados relacional para armazenamento persistente.
    OpenAPI 3.0 (Swagger): Documentação interativa e testes de endpoints em tempo real

## 🚀 Como executar

Clone o repositório e abra o arquivo `index.html` no navegador.

```bash
git clone https://github.com/infocbra/2025-1-noturno-pi3-g8-2026-1.git
```
API: O servidor embutido Tomcat iniciará automaticamente na porta 8080 após o comando de execução Maven ou Gradle. A documentação interativa estará disponível via Swagger UI logo após o startup da aplicação.
